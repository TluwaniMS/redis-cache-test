# redis-cache-test

# About Repository:
This is a basic node-js project that I created to learn how to cache API data using redis-cache.

# Software(s) required:
* Latest stable version of Node Package Manager
* MongoDB Database
* Redis-cache

* EXPRESS
* EXPRESS-GRAPHQL
* GRAPHQL
* MONGOOSE
* MORGAN

* DOTENV
